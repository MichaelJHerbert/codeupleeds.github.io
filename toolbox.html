Toolbox Links

This section is designed to give you information on the tools that can be used whilst you are not only learning about development but also as something you might use within your future career. I would recommend coming back to this section once you have further knowledge about the development process but you are also welcome to try them.

Google Chrome

Google Chrome is one of the most widely used browsers since its release in 2008 but the reason it is linked here is because of its abilty to use Developer tools, as you can see in the image below it allows you to access and change the code directly into the browser but will not be a permenant change, this means that you can make changes to elements of a website and it not affect your code directly. Google Chrome also has a console which you can write Javascript directly if you do not have something like visual studio code installed.

Visual Studio Code Extensions

These extensions are recommendations that will help aid you in development when you have gained a basic understanding of how to code, if you have installed visual studio code from here you can proceed through the installation process and on the left hand side click on the square icon at the bottom and proceed to search.

Live Server
