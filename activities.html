Github

Github, also known as Git is an open-source version control system and it is widely used by developers as a way to collaborate on projects. In short, it manages and stores each revision of your project as you make changes to the code on your local system you can then store a copy of it on Github, if there is a previous version of your code you would like to go back to at a later date.

If you would like to learn anything about Github and Git then feel free to ask one of the mentors or you can go to the Learning section to find out more.

#100 Days Of Code

Alexander Kallaway spent a year and a half learning to code and had many ups and downs along the way. He felt his experience would be useful to people and so set up #100DaysOfCode, challenging people to code for an hour a day. The challenge now has a small army of committed subscribers who share their projects, experiences and progress daily.
