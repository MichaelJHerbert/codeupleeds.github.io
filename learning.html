Learning Resources

A Full comprehensive list of resources that can be used to aid your learning ranging from free to paid courses and if you have one that is not listed below that you think can help others you can recommend it and we can add it below.
Additionally the premise of CodeUp UK and CodeUp Leeds is that you get help for free and the paid options provided below are an optional choice.

FreeCode Camp

A popular online learning tool for beginners to get to grips with new langauges and concepts. It offers training in different aspects of software and has challenges of varying difficulty once you're feeling confident once you are feeling confident with HTML/CSS or any other language you have chosen, I would recommend looking into the #100DaysOfCode.

Udemy

Udemy is an online platform for taking video tutorial courses from real-world professionals who want to teach others about their expertise, there is a wide range of courses not only limited to development and they have courses which are free with a range of offers which can get you courses for as little as Â£10, whilst the premise is to learn for free if you are interested we can always recommend some from this site to aid your learning.

Code Academy

Code Academy is similar to Free Code Camp in that it gives you specific problems to solve and as you solve each problem the difficulty level increases, you can learn about building websites, programming languages such as Python and introduction to Data Analysis. The idea behind Code Academy and Free Code Camp is to get you to a basic level so that you can proceed with a base level of understanding beyond that working on projects will help you further understand how programming is used.

Github

Since we already know what Github and Git are used for you can gain a better understanding on how to use Git and Github follow this link. A small note that Github is entirely free and if you choose you can add private repositories but this is a paid service.
