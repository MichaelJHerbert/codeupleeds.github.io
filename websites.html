Font Awesome

Want to add awesome fonts or images to your website? Font Awesome provides the tools for building apps on every type of Android device. Font Awesome is a font and icon toolkit based on CSS and LESS, the images you see on this site are provided by font Awesome.

Google Fonts

Want to add custom fonts to your website? Google Fonts provides a massive list of fonts for any website. It includes a layout editor, an emulator (to test your code without the hassle of transferring it to a phone/tablet), intelligent suggestions as you type, performance profilers and much more.
